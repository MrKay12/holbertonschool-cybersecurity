First repo for learn basic ssh stuff and the basics command.

Explication des commandes utilisees par exercice (sans prendre en compte les flags).

Exercice 0 - its_me
- whoami: affiche le nom de l'utilisateur courant.

Exercice 1 - find_complex
- find: recherche des fichiers/repertoires selon des criteres.

Exercice 2 - grep_secrets
- grep: recherche du texte qui correspond a un motif.

Exercice 3 - stats
- ls: liste le contenu d'un dossier.
- awk: extrait et transforme des colonnes de texte.
- sort: trie des lignes.
- uniq: fusionne les lignes identiques consecutives.
- head: garde les premieres lignes seulement.

Exercice 4 - suid_hunter
- find: commande deja expliquee dans l'exercice 1.

Exercice 5 - unlock
- chattr: modifie les attributs etendus d'un fichier.
- rm: supprime un fichier.

Exercice 6 - setup_shared
- mkdir: cree un repertoire.
- chown: change le proprietaire et/ou le groupe.
- chmod: change les permissions d'un fichier ou dossier.

Exercice 7 - audit_gateway
- cat: affiche le contenu d'un fichier, ici aussi utilise pour ecrire un fichier via redirection.
- chown: change le proprietaire et/ou le groupe.
- chmod: change les permissions d'un fichier ou dossier.

Exercice 8 - log_policy
- chown: commande deja expliquee dans l'exercice 6.
- chmod: commande deja expliquee dans l'exercice 6.
- echo: affiche une chaine, ici redirigee dans un fichier de configuration.
